Общие:
/spawn - С помощью данной команды вы можете вернуться на точку спавна сервера;
/sethome [Название дома] - Позволяет установить току дома (например: /sethome dom1);
/home [Название дома] - Команда позволяет телепортироваться в точку дома (например: /home dom1);

Операции с приватом:
//wand - Позволяет получить деревянный топор, с помощью которого можно осуществить выделение привата;
/region claim [Название региона] - Команда позволяющая создать приват;
/region delete [Название региона] - Команда позволяющая удалить приват;

Для VIP:
/nick - Данная команда открывает меню, для окраски ник-нейма;
/feed - Восполнение голода;
/back - Возврат в место последней смерти;
